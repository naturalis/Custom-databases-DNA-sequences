
# Installation

## Part 1: Downloading public sequence data of interest

### Requirements

* Python version 3.8.3

### Dependencies

The following libraries are used:

  * argparse 1.1
  * csv 1.0
  * Pandas 1.0.5
  * re 2.2.1
  * urllib3 1.25.9
  * taxonparser 0.2.3

## Part 2: Data classification & analyses

### Requirements
* R version 4.0.3

### Dependencies

The following packages are used:

  * rmarkdown 2.6
  * taxizedb 0.3.0
  * myTAI 0.9.2
  * tidyr 1.1.2
  * shiny 1.5.0
  * DT 0.17
  * plyr 1.1.2
  * dplyr 1.0.3
  * d3tree 0.2.2
  * billboarder 0.2.8



  
## Download the software

Clone the latest version from GitHub (recommended):  
`git clone https://github.com/naturalis/Custom-databases-DNA-sequences`  

Or, download the latest source codes:
* [Part 1](script/custom_databases.py)
* [Part 2](script/custom_databases.Rmd)
